# MyBook
- 교보문고를 모티브로 한 온라인 서점 서버 만들기 프로젝트 입니다.
- 백엔드에 집중하기 위해 프론트엔드 대신 Kakao Oven으로 만든 UI로 대신합니다.

## 사용 기술
- Spring Boot 2.7.6
- JAVA 11
- MyBatis 2.3.0
- MySQL
- Gradle
- Quartz
- Spring Security(Crypto)
- IntelliJ
- Postman
